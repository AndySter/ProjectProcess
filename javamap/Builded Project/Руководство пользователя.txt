Добро пожаловать в приложение по обработке изображений!
Данное приложение позволяет обрабатывать монохромные изображения со спутника, выделяя желтым цветом рукотворные/высокие объекты.

После запуска приложения пользователь может загрузить изображение 2 способами:
1. Нажать на кнопку "Загрузить" и выбрать на компьютере файл с нужным изображением;
2. Переместить на экран приложения файл нужным изображением.

Когда изображение будет загружено оно появится в блоке "Исходное изображение".
Для того чтобы обработать добавленное изображение, необходимо нажать на кнопку "Обработать".
После нажатия на кнопку "Обработать", обработанное изображение появится в блоке "Обработанное изображение".

Если пользователю необходимо изменить порог обработки выделения желтым цветом, то он может на ползунке "Порог" изменить критерии выделения.
Чем больше значения под ползунком, тем меньше объектов будет выделяться. Для того чтобы обновить результат, необходимо повторно нажать на
кнопку "Обработать".

После обработки изображения пользователю справа в верхнем углу демонстрируется время обработки изображения в миллисекундах.

Так же пользователь может приближать/отдалять изображения для их сравнения. Чтобы сделать это, необходимо двигать ползунок "Приблизить".
Чем больше значение под ползунком, тем ближе будут изображения. Рекомендуется использовать ползунок,  когда будет показано обработанное изображение.

Если изображения больше экрана приложения, то есть возможность перетаскивать их, чтобы видеть изменения в разных местах изображения.

Если необходимо сохранить обработанное изображение, то пользователю необходимо нажать на кнопку "Сохранить". Тогда в открывшемся окне пользователь
сможет сохранить изображение в любое место на компьютере в файл формата PNG.
